# Ponpon

Finger based adaptation of popular Pong! game using medipipe for hand tracking

# External Dependencies

- `direnv` (added for automatic sourcing when opening project)
- `poetry` (for dependencies management)

# Run the project

1. Clone the repo with `git clone`
2. `cd` into the cloned folder and run `direnv allow`
3. Run `poetry install` to install project dependencies
4. After everything is done run `poetry run start` to start the game
